Curso: Programación I

El curso de Programación I constituye la base esencial para el desarrollo de habilidades en lógica computacional
y resolución de problemas mediante lenguajes de programación estructurados. A lo largo del curso, el estudiante aprende a diseñar
algoritmos y traducirlos en código ejecutable, usando estructuras de control como condicionales, bucles y funciones. Se trabaja
con tipos de datos, operadores, arreglos, manejo de entradas y salidas, así como buenas prácticas de codificación. El objetivo
principal es que el estudiante sea capaz de construir programas funcionales, estructurados y claros, comprendiendo cómo se
ejecutan las instrucciones dentro de un flujo lógico. Este conocimiento sienta las bases para cursos avanzados como Programación
Orientada a Objetos, Estructuras de Datos y Desarrollo de Software.

Puntos Clave

  • Desarrollo de lógica algorítmica mediante pseudocódigo y diagramas de flujo.
  • Uso de estructuras de control de flujo: condicionales (if-else, switch) y ciclos (while, for, do-while).
  • Declaración, inicialización y manipulación de variables y constantes de distintos tipos de datos.
  • Trabajo con operadores aritméticos, lógicos y relacionales para construir expresiones.
  • Manejo de arreglos unidimensionales para almacenar y procesar datos en colecciones.
  • Modularización del código mediante el uso de funciones, argumentos y retorno de valores.
  • Lectura y escritura de datos desde el teclado y hacia la pantalla (entrada/salida básica).

Temas del Curso

Tema Descripción
Introducción a la Programación Conceptos básicos de programación, algoritmos, lenguajes y compiladores.
Tipos de Datos y Variables Enteros, flotantes, caracteres, cadenas, booleanos. Declaración y asignación de variables.
Operadores Uso de operadores matemáticos, lógicos y de comparación en expresiones.
Estructuras de Control Condicionales (if-else, switch) y ciclos (for, while, do-while) para controlar el flujo del programa.
Funciones Modularización del código usando funciones con paso de parámetros y valores de retorno.
Arreglos Uso de arreglos unidimensionales para procesar múltiples valores del mismo tipo.
Entrada y Salida Lectura de datos desde el teclado y despliegue de resultados en pantalla.
Buenas Prácticas Indentación, nombres descriptivos de variables, comentarios y estructura legible.
